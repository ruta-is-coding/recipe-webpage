# tailwind installation

choose a folder->
open terminal->
npm init -y + enter->
npm install tailwindcss + enter->
npm run build-css + enter

# tailwind configuration

open a new terminal->
npx tailwindcss init --full + enter (for a default config file)->
npx tailwindcss init + enter (for a blank config file)

# changing config file (adding new keys inside an extend object)

if I change a tailwind.config.js file->
npm run build-css
